# MIUH - Plataforma de Monitoreo y Alerta de Inundaciones

Proyecto académico desarrollado en la Universidad Continental.

##  Descripción
MIUH es una plataforma web que permite monitorear en tiempo real los niveles de agua en zonas críticas de Huancayo (Chilca y El Tambo), y emitir alertas a la comunidad.

##  Tecnologías utilizadas
- Frontend: Angular
- Backend: Node.js + Express
- Base de datos: MySQL
- Control de versiones: Git y GitHub
